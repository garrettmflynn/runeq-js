To generate documentation for `runeq`, run:
```basn
npm run start
```

## Usage
### Browser
#### Script Tag
```html
<script src="https://cdn.jsdelivr.net/npm/runeq@latest"></script>
```

#### ES6 Module
```js
import * as runeq from 'https://cdn.jsdelivr.net/npm/runeq/dist/index.esm.js'
```

### Node
```basn
npm install runeq
```
